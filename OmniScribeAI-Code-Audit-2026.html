8. AI/ML Pipeline

7-Pass Note Generation (SSE Streaming)

Audio → Deepgram Nova-3-Medical → Transcript | Pass 0: PHI De-identification (Claude) Pass 1: Fact Extraction (DeepSeek/Claude/Grok) └── CMS Compliance Scoring (deterministic, 708 items) Pass 2: Clinical Synthesis (Claude) Pass 3: Parsed Data (Claude) Pass 4: Final Clinical Note (Claude + PHI re-identification) Pass 5: Hallucination Audit (Claude) Pass 6: Summary (Claude)

Fast 2-Pass Mode (Real-Time Encounter State)

Recording → VAD + 10s chunks → Groq Whisper → LLM extraction | EncounterState (live) | Pass 1: Generate note from validated facts (Claude) Pass 2: Audit + summary (Claude)

2-Pass OCR Pipeline

Image/PDF → Claude Vision → Text Extraction → Clinical Mapping → Structured JSON

9. HIPAA Security Architecture

  • PHI Endpoint Whitelist — all external API calls validated against approved endpoints
  • PHI De-identification — Pass 0 strips all PHI before sending to AI providers; re-identified only in final output
  • Client-side PHI Isolation — separate storage namespace via phi-storage.ts
  • 8-Hour JWT Sessions — with forced extended session disclaimer acknowledgment
  • Forced Password Change — mustChangePassword flag for new users
  • Middleware Auth Enforcement — all routes protected except /login, /api/auth, /
  • Audit Logging — all clinical actions logged to AuditLog table (user, action, IP, user agent)
  • Password Hashing — bcryptjs
  • Session Tracking — lastLoginAt, extendedSessionAcknowledgedAt
