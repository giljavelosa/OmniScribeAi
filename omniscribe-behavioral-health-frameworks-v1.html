How to Use These Templates

  1. Select the Framework — Choose from Psychiatric Evaluation, Therapy Notes, or Treatment Review/Discharge based on the clinical encounter type.
  2. Choose the Subtype — Each framework has 5 specialized subtypes for different clinical scenarios.
  3. Use the Format Templates — Each item provides a structured format string with labeled fields (marked with __) ready for clinical data entry.
  4. Reference the Sources — Source citations identify the regulatory basis for each documentation element.
  5. Customize as Needed — Templates can be adapted to your EHR, payer requirements, and state-specific regulations.

Note: These templates are designed to meet the most stringent documentation standards across accreditation bodies. Not all items are required for every encounter — select those relevant to your clinical setting, patient population, and payer requirements.
